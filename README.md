# 📡 MEXC Lag Bot

Відстежує **затримку MEXC** відносно Binance, Bybit, OKX, Gate, BingX.
Показує тільки токени де **реально є** затримка (5+ підтверджень).

---

## 🚀 Деплой на Vercel (5 хвилин)

### Варіант 1 — через GitHub (рекомендовано)
1. Створіть репозиторій на GitHub і завантажте всі файли
2. Зайдіть на [vercel.com](https://vercel.com) → **New Project**
3. Виберіть ваш репозиторій → **Deploy**
4. Готово! Vercel сам зробить все інше

### Варіант 2 — Vercel CLI
```bash
npm install -g vercel
cd mexc-lag-bot
vercel --prod
```

---

## 💻 Локальний запуск
```bash
npm install
npm run dev
# Відкрийте http://localhost:3000
```

## ⚙️ Як це працює

```
Binance/Bybit/OKX/Gate/BingX  ──WebSocket──▶  Ціна рухнулась
                                                      │
                                               Записуємо час T1
                                                      │
MEXC  ──────────────────WebSocket──▶  Ціна рухнулась T2
                                                      │
                               Lag = T2 - T1 = 2.4с ✅
```

**Токен з'являється** на дашборді тільки якщо:
- Зібрано ≥ 5 збігів руху
- Середня затримка MEXC ≥ 500мс
- MEXC має нульову комісію на цей токен

---

## 📊 Що показує кожна картка

| Поле | Опис |
|------|------|
| **Затримка** | Середня затримка MEXC у секундах |
| **min/max** | Мінімальна та максимальна затримка |
| **Прибуток 100×** | Очікуваний % при 100x плечі (без урахування ліквідації) |
| **Біржі** | Які біржі найчастіше лідирують (%) |
| **Зразків** | Кількість підтверджених збігів |

## ⚠️ Важливо

- **Vercel Hobby** має ліміт 300с на serverless функцію — SSE stream може обриватись

- Vercel підходить для тестування та демонстрації
